## Deployment (GitHub Pages)

1. Create a repo named `tahirhusen.github.io` (for username site) or any repo name
2. Upload `index.html` and `README.md` to root
3. Optionally add `resume.pdf` for the resume download link
4. Go to **Settings → Pages → Deploy from branch → `main` → `/ (root)`**
5. Site will be live within minutes

### Custom Domain (Optional)

To connect a custom domain later, add a `CNAME` file with your domain name.

## Tech Stack

- Single self-contained HTML file (50KB)
- Zero dependencies, no build step
- Embedded base64 profile photo
- CSS-only animations and scroll-reveal effects
- Responsive design (desktop, tablet, mobile)
- Fonts: Cormorant Garamond + Darker Grotesque (Google Fonts)

## To Customize

- **Calendly link**: Replace `https://calendly.com/tahirchisti` with your actual Calendly URL
- **Resume PDF**: Add a `resume.pdf` file to the repo, then update the resume download link
- **Dashboard URLs**: Update portfolio demo links to your actual GitHub Pages URLs
- **Photo**: Embedded as base64 — to change, replace the `data:image/png;base64,...` in the `<img>` tag
